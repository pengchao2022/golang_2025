
Go 语言中整形 int8

int8 是带有符号的 取值范围为： -128 - 127

uint8 是无符号的 取值范围为： 0 - 255

int16 的取值范围 -32768 到 32767

uint16 的取值范围 0 到 65535

int32  的取值范围 -2147483648 到 2147483647

uint32 的取值范围 0 到 4294967295

int64 的取值范围 -9223372036854775808 到 9223372036854775807

int 与平台相关， 32 位操作系统上是 int32 , 64 位操作系统上是 int64

Go 语言中 浮点型  float32 和 float64

float64 能保留更多的小数位
